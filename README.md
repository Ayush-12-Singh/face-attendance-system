# Face Recognition Attendance System

An AI-based attendance tracking dashboard built for J-Techtrix 7.0. This system uses facial recognition to automate student attendance, featuring a web dashboard for live tracking and data export.

## Features
* **Live Face Detection:** Uses OpenCV and an LBPH recognizer to identify students in real-time.
* **Web Dashboard:** A Flask-based frontend to view and manage student profiles.
* **Excel Export:** Easily export daily or subject-wise attendance records to `.xlsx`.
